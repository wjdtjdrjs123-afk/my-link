# 사용자 시나리오 (User Scenario) : 마이 링크(My Link)

본 문서는 마이 링크 서비스를 이용하는 모든 사용자의 제품 내 경험 흐름을 구체적인 행동 기반으로 작성한 시나리오 명세서입니다. 사용자는 **방문자(Visitor)**와 프로필을 보유한 **소유자(Owner)** 두 가지 역할로 구분됩니다.

---

## 1. 방문자 (Visitor) 시나리오

마이 링크 구조 상, 방문자는 별도의 소셜 기능 없이 오로지 화면을 조회하고 타겟 링크로 이동하는 직관적인 활동에 집중합니다.

*   **1-1. 페이지 접속**
    *   사용자는 개발자나 크리에이터의 포트폴리오(또는 활동 내역)를 모아서 보기 위해 공유받은 링크 URL(`mylink.com/닉네임`)을 클릭하여 해당 프로필 페이지에 접속한다.

*   **1-2. 기본 프로필 확인**
    *   사용자는 현재 접속한 페이지가 누구의 홈페이지인지 파악하기 위해 화면 최상단에 있는 닉네임 문자열과 간단한 소개글(Bio)을 읽는다.

*   **1-3. 링크 탐색 및 시각적 인지**
    *   사용자는 자신에게 필요한 외부 연결 사이트(예: 깃허브, 개인 블로그 등)를 찾기 위해, 아이템 목록들을 스크롤업/다운하며 링크 박스 좌측에 렌더링된 파비콘(Favicon) 이미지 피드를 통해 빠르게 직관적으로 훑어본다.

*   **1-4. 링크 클릭 및 조회수 기여**
    *   사용자는 원하는 외부 웹사이트로 이동하기 위해 해당 사이트의 링크 버튼 블록을 클릭한다. (이때 백그라운드에서는 소유자 통계 데이터에 반영하기 위해 해당 링크 아이템의 클릭 트래킹 카운터가 1 증가한다.)

---

## 2. 소유자 (Owner) 시나리오

프로필 페이지를 소유하고 관리하는 크리에이터가 로그인 및 온보딩부터, 정보 갱신, 링크 제어에 이르는 모든 관리 과정을 수행합니다.

### 2-1. 온보딩 (회원가입 및 로그인)

*   **구글 소셜 로그인**
    *   사용자는 서비스에 복잡한 정보 입력 없이 가입하기 위해 초기 화면에서 'Continue with Google' 로그인 버튼을 클릭한다.
*   **초기 정보 자동 로드**
    *   사용자는 가입과 동시에 본인만의 개별 접속 주소(`mylink.com/닉네임`)를 획득하기 위해, 구글 계정 아이디(메일주소 앞부분) 기반으로 자동 세팅된 접속 URL 닉네임 값을 화면에서 확인한다.

### 2-2. 프로필 정보 갱신 (인라인 편집)

*   **닉네임 수정**
    *   사용자는 웹브라우저 상단이나 사람들에게 복사해서 넘겨주는 자신의 URL 슬러그를 변경하기 위해, 화면에 크게 적힌 닉네임 텍스트를 마우스로 직접 클릭한다. 텍스트가 인라인 입력창으로 바로 폼 체인지 되면 원하는 영문/숫자를 새롭게 입력하고 포커스 아웃(또는 엔터키)하여 정보를 즉시 덮어쓴다.

*   **소개글(Bio) 수정**
    *   사용자는 페이지 방문객에게 스스로를 설명하기 위해, 기존의 짧은 글귀(또는 '소개를 입력해주세요' 플레이스홀더) 부분을 클릭한 뒤 나오는 인라인 입력 창에 커리어나 작업 안내 텍스트를 적고 저장한다.

*(참고: 이미지가 아닌 텍스트 기반으로만 심플하게 나를 표현하기 위해 별도의 프로필 사진 업로드 단계는 진행하지 않는다.)*

### 2-3. 링크 아이템 관리 (CRUD 인라인)

*   **빈 상태 (Empty State) 인지**
    *   사용자는 처음 가입하여 아이템이 단 하나도 등록되지 않았을 때 무엇을 해야 할지 파악하기 위해 화면 중앙에 뜬 '첫 번째 링크를 추가해 보세요!'라는 안내 텍스트를 확인한다.

*   **신규 링크 블록 생성 (Create)**
    *   사용자는 방문자들에게 새 웹사이트를 공유하기 위해, 화면 상단 또는 빈 리스트 하단에 위치한 **'+ 새 링크 추가'** 버튼을 클릭하여 비어 있는 새로운 링크 폼 블록을 화면에 생성시킨다.

*   **링크 정보 인라인 기입 및 자동 파비콘 (Update & Extract)**
    *   사용자는 생성된 빈 상자에 실제 도착 주소와 제목을 연결시키기 위해 생성된 상자 내부의 `[제목을 입력하세요]` 부분을 직접 인라인 클릭하여 텍스트를 입력한다. 같은 방법으로 `[URL을 입력하세요]` 부분도 클릭 후 실제 링크 주소를 적어 엔터를 누른다.
    *   사용자는 자신이 올바른 링크 도메인을 입력했는지 확인하기 위해, 입력 직후 우측에서 자동으로 불러와 교체되는 파비콘 이미지를 육안으로 확인한다.

*   **불필요한 링크 삭제 (Delete)**
    *   사용자는 기간이 지났거나 더 이상 쓰지 않는 사이트로의 유입을 막기 위해 굳이 비활성화하지 않고, 링크 블록의 우측 가장자리에 고정된 **휴지통 아이콘(🗑️)**을 클릭하여 페이지에서 링크 박스 전체를 즉각적이고 영구적으로 걷어낸다.
